# unit-converter
this app just for learning python

A simple, friendly command-line tool for common unit conversions.


- m → cm,
- m → km,
- kg → lb
- °C → °F

Example

Options
1) m to cm
2) m to km
3) m to ft
4) kg to pound
5) °C to °F
0) Exit
--------------------
Enter chosen option: 3

Enter the m number to ft: 190
190 m = 623.36 ft
